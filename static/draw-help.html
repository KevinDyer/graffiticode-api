Canvas properties
  • background color
  • Set the color of the drawing surface to color.
  • size width height
  • Set the size of the drawing surface to width and height.
Constructor functions
  • bezier x1 y1 x2 y2 x3 y3 x4 y4
  • Construct a bezier curve that starts at point x1, y1 and ends at x4, y4 with intermediate control points x2, y2 and x3, y3.
  • ellipse dx dy
  • Construct an ellipse with horizontal and vertical diameters dx and dy with center at the origin.
  • grid dx dy width height
  • Construct a grid with x and y mesh size dx and dy and covering the rectangle width and height with top left corner at the origin.
  • rectangle width height
  • Construct a rectangle with width and height and top left corner at the origin.
  • text str
  • Construct a text of string str positioned with its bottom left corner at the origin.
  • triangle x1 y1 x2 y2 x3 y3
  • Construct a triangle with points (x1, y1), (x2, y2), (x3, y3).
  • triside l1 l2 l3
  • Construct a triangle with sides of length l1, l2 and l3 positioned at the origin.
Modifier functions
  • Modifier functions update and return the value of the last argument. Returning the modified value allows modifier functions to be chained.
  • color color val
  • color represents a stroke and fill color. val is the value to be re-colored. If color is an integer, use the RGB value that results by calling rgb color color color.
  • fill color val
  • color represents a color value. val is the value to be re-colored. If color is an integer, use the RGB value that results by calling rgb color color color.
  • font-size size val
  • size is a font size in pixels. val is a text value to be resized.
  • rotate angle val
  • angle is the rotation angle. val is the value to be rotated.
  • scale factor val
  • factor is the scale factor. val is the value to be scaled.
  • skewx angle val
  • angle is the rotation angle of the x-axis. val is the value to be skewed.
  • skewy angle val
  • angle is the rotation angle of the y-axis. val is the value to be skewed.
  • stroke color val
  • color is the color to be used. val is the value to be recolored. If color is an integer, use the RGB value that results by calling rgb color color color.
  • translate x y val
  • x and y are the x and y distance moved. val is the value to be moved.
Operators
  • val1 add val2
  • Return the sum of the numeric values val1 and val2.
  • val1 div val2
  • Return the result of dividing numeric values val1 by val2.
  • val1 mul val2
  • Return the result of multiplying the numeric values val1 by val2.
  • val1 sub val2
  • Return the result of subtracting the numeric values val2 from val1.
Value generators
  • random min max
  • Return a psuedo-random integer between the values of min and max, inclusive.
  • rgb r g b
  • Return a RGB value composed of the values of r, g and b where each argument is an integer value between 0 and 255, inclusive.
  • rgba r g b a
  • Return a RGBA value composed of the values of r, g, b and a. where each argument r, g, and b is an integer value between 0 and 255, inclusive, and a is an integer value representing the transparency between 0 and 100, inclusive, where 0 denotes invisible and 100 denotes opaque.
