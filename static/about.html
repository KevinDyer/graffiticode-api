About Graffiti Code


Graffiti Code is a coding environment for synthesizing form. It is experimental and currently hard to decode as no documentation is provided. If you are curious, you can click around the Archive to see some of our experiments. If you are feeling lucky, you can login with Persona and try coding some yourself. Beware though, the only help we can offer is the hints you can gleen from the code in the archive. Oh, and we recommend using an recent version of Firefox.
Graffiti Code is produced by Art Compiler .
