Getting started
  • Starting, saving and sharing
  • The buttons above the editor allow you to start a new program, fork an archived program, save a program to the archive, and share a program on Github.
  • Writing code
  • A Graffiti Code program is composed of expressions. There are several kinds of expressions in Graffiti Code. Most expressions have the form of a function name followed by a fixed number of arguments. Functions that create values, such as rectangle, are called constructor functions. Functions that modify property values, such as color, are called modifier functions. Modifier functions return the value they modify, allowing for nesting of modifiers in a single expression.

    Expressions that are not nested within other expresssions are terminated with a period. A list of expressions is terminated by another period. You will notice that there are two periods at the end of a program and other expression lists such as the body of a let expression.

    New function names are defined by let expressions. Let expressions capture the lexical scope with a map of parameter names followed by its outer scopes. The body of a let expression is a list of expressions separated by periods and terminated by two periods.

    In addition to call and let expressions, Graffiti Code includes binary expressions and prefix and postfix unary expressions.

    Also, Graffiti Code provides syntax for integer, string, boolean, tuple, list and map values.
Canvas properties
  • background rgb
  • Set the canvas color with the RGB value rgb.
  • size width height
  • Set the canvas size with two integer values width and height.

Constructor functions
  • bezier x1 y1 x2 y2 x3 y3 x4 y4
  • A bezier curve that starts at point x1, y1 and ends at x4, y4 with intermediate control points x2, y2 and x3, y3.
  • ellipse rx ry
  • An ellipse with horizontal and vertical diameters dx and dy with center at the origin.
  • grid dx dy w h
  • A grid with x and y mesh size dx and dy and covering the rectangle w and h with top left corner at the origin.
  • rectangle w h
  • A rectangle with width w and height h and top left corner at the origin.
  • text str
  • A text of string str positioned with its bottom left corner at the origin.
  • triangle x1 y1 x2 y2 x3 y3
  • A triangle with points (x1, y1), (x2, y2),
    (x3, y3).
  • triside l1 l2 l3
  • A triangle with sides of length l1, l2 and l3 positioned at the origin.

Modifier functions
  • Modifier functions update and return the value of the last argument. Returning the modified value allows modifier functions to be chained.
  • font-size size text
  • Change the font size of text.
  • translate dx dy val
  • Update the position of val by adding dx and dy to its current x and y values, respectively. Return the modified value of val.

  • Change the size of val by scaling it by factor n.

Operators
  • val1 add val2
  • Return the result of adding val1 and val2.
  • val1 div val2
  • Return the result of dividing val1 by val2.
  • val1 mul val2
  • Return the result of multiplying val1 by val2.
  • val1 sub val2
  • Return the result of subtracting val2 from val1.
