Canvas properties
  • background rgb
  • Set the canvas color with the RGB value rgb.
  • size width height
  • Set the canvas size with two integer values width and height.
Constructor functions
  • bezier x1 y1 x2 y2 x3 y3 x4 y4
  • A bezier curve that starts at point x1, y1 and ends at x4, y4 with intermediate control points x2, y2 and x3, y3.
  • ellipse rx ry
  • An ellipse with horizontal and vertical diameters dx and dy with center at the origin.
  • grid dx dy w h
  • A grid with x and y mesh size dx and dy and covering the rectangle w and h with top left corner at the origin.
  • rectangle w h
  • A rectangle with width w and height h and top left corner at the origin.
  • text str
  • A text of string str positioned with its bottom left corner at the origin.
  • triangle x1 y1 x2 y2 x3 y3
  • A triangle with points (x1, y1), (x2, y2),
    (x3, y3).
  • triside l1 l2 l3
  • A triangle with sides of length l1, l2 and l3 positioned at the origin.
Modifier functions
  • Modifier functions update and return the value of the last argument. Returning the modified value allows modifier functions to be chained.
  • font-size size text
  • Change the font size of text.
  • translate dx dy val
  • Update the position of val by adding dx and dy to its current x and y values, respectively. Return the modified value of val.

  • Change the size of val by scaling it by factor n.

Operators
  • val1 add val2
  • Return the result of adding val1 and val2.
  • val1 div val2
  • Return the result of dividing val1 by val2.
  • val1 mul val2
  • Return the result of multiplying val1 by val2.
  • val1 sub val2
  • Return the result of subtracting val2 from val1.
