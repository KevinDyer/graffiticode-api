03 DEC 2012

Preview Release

Graffiti Code is currently available as a Preview Release. This means that it is suitable for doing demos and using along well documented paths.

The greatest disappointments in this release will probably have to do with performance, unimplemented language features, and lack of archive filtering. We sugguest that you start small, stay within the documented path, and be patient scrolling through however many pieces are found in the archive.

Send mail to feedback@graffiticode.org if you have an issue you want prioritied.

Happy coding!
